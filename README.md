# Workout Tracker

A mobile application for tracking workouts and logging sets, reps, and weights. Data is stored locally using SQLite, providing persistence across app restarts.

## Features:

* Add workouts and exercises
* Log sets with weight and reps
* Swipe to delete or long-press to edit entries
* SQLite-backed data persistence
* Modal-based UI and navigation

## Status:

MVP currently in development.
